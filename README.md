# Employee Payroll System

## Description
This is a Java console application that calculates an employee's salary.

## Features
- Accepts Employee Name
- Accepts Employee ID
- Accepts Basic Salary
- Calculates HRA
- Calculates DA
- Calculates PF
- Calculates Gross Salary

## Technologies Used
- Java
- Scanner Class

## How to Run

Compile:
javac EmployeePayroll.java

Run:
java EmployeePayroll
